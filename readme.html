Word Jumble

Drag the letters in the grid above to the blank white area below, arrange them to make a word and keep unused letters in the original grid. Then click the box area to check the word and see if it is a real word. The API used is https://dictionaryapi.dev/

Puzzle Game

Solve the puzzle by moving the pieces into the correct spot, Choose a difficulty setting and load the puzzle The hint button will show the Image but be careful, it wont stay up there long. Moving the mouse will cause it to disappear.

Car Obstacle Game

Use the arrow keys on your keyboard or the keys below to move the Unit around to avoid the obstacles

Left Key: Moves the unit Left

Right Key: Moves the unit Right

Up Key: Moves the unit Up

Down Key: Moves the unit down
